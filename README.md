# Bean & Brew — Coffee Shop Website

A responsive, static website for **Bean & Brew**, a neighborhood coffee shop. The site presents the brand, welcoming copy, and a full menu of coffee and pastries in a clean, accessible layout.

## Overview

This project delivers a small multi-page website built with HTML and CSS. It is suitable for local hosting, portfolio use, or as a base for a café or similar business site.

## Project Structure

```
.
├── index.html      # Home page — welcome and introduction
├── menu.html       # Menu page — coffee selection and pastries
├── style.css       # Global styles and layout
├── images/         # Logo and menu imagery (coffee, pastries)
└── README.md       # This file
```

## Features

- **Home page** — Hero section, brand message, and brief introduction to the shop
- **Menu page** — Coffee drinks and fresh pastries with prices; image carousels for visual appeal
- **Responsive layout** — Styling adapts to different screen sizes
- **Typography** — Google Fonts (Amatic SC, Quicksand) for a consistent, readable look
- **Semantic HTML** — Header, main, section, and footer for structure and accessibility

## Technologies

- **HTML5** — Structure and content
- **CSS3** — Layout, typography, colors, and responsive behavior
- **Google Fonts** — Web fonts loaded via CDN

## Getting Started

1. **Clone or download** this repository to your machine.
2. **Open in a browser**  
   - Double-click `index.html`, or  
   - Use a local server (e.g. `python -m http.server` or Live Server in your editor) and open the URL given (e.g. `http://localhost:8000`).
3. Use the navigation to move between **Home** and **Menu**.

No build step or external backend is required; the site runs as static files only.

## Browser Support

The site uses standard HTML and CSS and is intended to work in current versions of major browsers (Chrome, Firefox, Safari, Edge).
